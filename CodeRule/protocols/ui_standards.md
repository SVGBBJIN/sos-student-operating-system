# Protocol: UI & UX Standards

## Visual Identity
- **Color Palette:** - Primary: High-Tech Blue (`#001219`)
  - Accent: Electric Cyan (`#00E5FF`)
  - Warning: Neon Purple (`#7209B7`)
- **Typography:** Use 'Inter' for body and 'JetBrains Mono' for code snippets or technical labels.
- **Components:** - Buttons should have a subtle glow effect on hover using Cyan.
  - Containers must have a 1px border of `rgba(0, 229, 255, 0.2)` to look like a HUD.

## Interactive Gaps
- **Loading States:** Always use a skeleton loader or a pulsing "scanning" animation.
- **Feedback:** Every button click must provide immediate visual feedback (active state change).
- **Responsiveness:** Build mobile-first. Desktop views should never feel "stretched."
